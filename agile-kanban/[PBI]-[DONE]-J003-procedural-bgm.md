# PBI-J003: Procedural Background Music

## Description
Generate atmospheric background music using Web Audio API oscillators and patterns. Different tracks for menu, combat, and boss encounters.

## Acceptance Criteria
- [ ] Menu BGM: Calm, atmospheric Chinese/oriental-inspired drone
- [ ] Combat BGM: Driving percussion-based beat with intensity scaling
- [ ] Boss BGM: Intense variation with faster tempo
- [ ] Music volume control separate from SFX
- [ ] Smooth crossfade between tracks on state change
- [ ] Mute toggle respects existing audio toggle

## Estimate: Medium (45-60 min)
