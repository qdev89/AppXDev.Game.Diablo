# [PBI] H005 - Pause Menu & Settings

## Description
Add Escape key pause menu during gameplay:
- Pause/Resume with ESC key
- Settings: Language toggle (VI/EN), SFX volume, Music toggle
- Show current hero stats summary
- "Return to Menu" option
- Dark overlay with centered panel

## Acceptance Criteria
- [ ] ESC key pauses game and shows overlay menu
- [ ] Resume button or ESC again unpauses
- [ ] Language toggle works from pause menu
- [ ] Return to Menu option works
- [ ] Game state frozen while paused (no enemy movement, no timers)
