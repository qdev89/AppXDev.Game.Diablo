# PBI-J002: Main Menu Animation

## Description
Polish the main menu with animated particles, floating elements, and dynamic title effects to create a premium first impression.

## Acceptance Criteria
- [ ] Animated particle effects on title screen (embers/sparks)
- [ ] Pulsating/glowing title text
- [ ] Floating warrior silhouettes or weapon icons in background
- [ ] Smooth hover/select feedback on menu items
- [ ] "Press to Start" text with breathing animation

## Estimate: Medium (30-60 min)
