# PBI-002: High Score Leaderboard

## Description
Add localStorage-based high score system. Show top 5 scores on menu screen. Track score, floor, level, grade, and timestamp.

## Acceptance Criteria
- [ ] Save run stats to localStorage on game over
- [ ] Display top 5 scores on menu screen
- [ ] Each entry shows: Score, Floor, Grade, Date
- [ ] New high score highlight animation
- [ ] Zero JS errors

## Estimate: Small (20min)
