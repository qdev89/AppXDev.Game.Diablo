# PBI-J004: LocalStorage Persistence

## Description
Save game progress, high scores, selected language, Arcana upgrades, and settings to localStorage for persistence across browser sessions.

## Acceptance Criteria
- [ ] Save high score (best floor reached, total kills) on game over
- [ ] Save selected language preference
- [ ] Save audio on/off preference
- [ ] Save Arcana skill tree progress
- [ ] Load saved data on game startup
- [ ] Display personal best on results/menu screen

## Estimate: Small (20-30 min)
